Players earn Command Points during their Alt-Hammer battles, which are spent on Stratagems that can provide situational advantages for your army. The rules for gaining Command Points, spending them on Stratagems, and the available Stratagems are contained here.

### Gaining Command Points

Command Points are gained during Step 2 of the Command Phase. Each player gains a number of Command Points based on the following: 
1. 1 new Command Point at a minimum
2. 1 additional Command Point for each Leader unit currently on the battlefield (including those embarked in a Transport).
3. 2 additional Command Points if your Warlord is positioned on a Primary Objective Marker that you control either in No Man's Land or in the enemy's Deployment Zone.

### Spending Command Points

Each Strategem identifies the number of Command Points needed to use that Stratagem. Some Stratagems used to impact a unit's Activation have different Command Point costs based on the type of unit - friendly or enemy - that you choose to use that Stratagem for.
